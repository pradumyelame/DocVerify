# Intelligent Document Verification System (IDVS)

An automated system designed to verify the authenticity of documents through advanced image preprocessing and multiple feature extraction techniques. The system aims to detect forgeries, tampering, and ensure document integrity.

## 🚀 Key Features

- **21 Advanced Preprocessing Techniques**: Includes Adaptive Thresholding, Otsu, CLAHE, Morphological operations, and more.
- **Blockchain Data Integrity**: Every text extraction is cryptographically hashed and stored in a local immutable ledger to prevent tampering.
- **Multiple Feature Extraction**: Utilizes CNN, SIFT, HOG, LBP, and ORB.
- **Secure Dashboard**: Streamlined interface focused on direct document analysis results.

## 🛠️ Tech Stack

- **Frontend**: React.js (Vite), Axios, Lucide React, CSS3.
- **Backend**: Python (Flask), OpenCV, NumPy, Flask-CORS, **Hashlib (Blockchain)**.
- **Database**: SQLite & **Local Blockchain Ledger** (`blockchain.json`).

## ⚙️ Setup & Installation

### Prerequisites

- Python 3.8+
- Node.js (Latest LTS recommended)
- npm or yarn

### Backend Setup

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Create and activate a virtual environment:
   ```bash
   python -m venv .venv
   # Windows
   .venv\Scripts\activate
   # macOS/Linux
   source .venv/bin/activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the backend server:
   ```bash
   python app.py
   ```
   The server will start at `http://localhost:5000`.

### Frontend Setup

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
   The application will be accessible at `http://localhost:5173`.

## 📖 Usage

1. Start both the backend and frontend servers.
2. Open the frontend application in your browser.
3. Upload a document image for verification.
4. Select the desired preprocessing and feature extraction techniques.
5. Review the verification report generated by the system.
